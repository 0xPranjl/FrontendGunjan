# 🏔️ Ultra-Premium Interactive Wall Calendar

A masterpiece of frontend engineering that brings a physical wall calendar to life in a digital space. Designed for visual impact and technical excellence.

## 🏆 Key Features

- **Physical World Realism**: Realistic wall environment with texture, directional lighting, and a 3D hanging wire/nail effect.
- **Modern Badge Notes**: Notes section redesigned as a vibrant blue badge cloud with interactive stickies and smooth state transitions.
- **Precision Navigation**: Direct Month and Year selection via custom-styled selectors, plus traditional 3D "flipping" navigation.
- **Cinematic Animations**: Spring-based 3D entry, stagger-loaded grids, and parallax hero imagery.
- **Data Persistence**: Instant LocalStorage sync for all notes and badges.

## 🚀 Get Started

1. `npm install`
2. `npm run dev`
3. View the masterpiece at `http://localhost:5173`.

## 🛠 Built With

- **React 18**: UI Logic.
- **Framer Motion**: Physical-grade animations.
- **Lucide Icons**: Premium vector assets.
- **Custom CSS Modules**: For zero-conflict, high-performance styling.
